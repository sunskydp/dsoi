T202835 账号管理系统

题目描述

fj有许多的账号, 但他总是记不住, 于是找到了学信奥的你, 请你帮他写一个账号管理系统。

输入格式

第一行输入一个数 N 和 M, 分别表示fj的账号数量和小A的操作数量。

接下来 N 行, 每行用空格隔开的两个字符串 Di 和 Pi, 分别表示账号和密码。

再接下来 M 行, 每行一个数 Fi 和一个字符串 Si, 分别表示fj操作的编号和要操作的账号。

操作编号只有 1 和 2, 操作方法具体如下：

操作编号1：查询账号为 Si 的密码。

操作编号2：删除账号为 Si 的密码。

输出格式

对于每一个操作的返回输出如下:

操作编号1：如果找到了,输出其对应的密码;如果没找到, 输出“Not found.”。

操作编号2：如果有这个账号,输出“Completed.”;如果没找到, 输出“Error.”。

输入样例 #1

2 2

xiao_A password

xiao_ming 13123AU09

1 xiao_A

2 xiao_A

输出样例 #1

password

Completed.

说明/提示

n ≤ m ≤ 100

Di ≤ Pi ≤ Fi ≤ Si ≤ 231
